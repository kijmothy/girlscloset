# Girls Closet Official Website v1.0 Final

這是女孩衣櫃間 Girls Closet 的官方網站 v1.0 Final。

## 檔案結構

- index.html：網站主要內容
- style.css：網站樣式
- logo.png：品牌 Logo
- images/products/：商品圖片資料夾

## 更新方式

1. 修改本機資料夾中的 index.html、style.css 或圖片。
2. 到 GitHub repository：girlscloset。
3. 點 Add file → Upload files。
4. 拖曳更新後的檔案。
5. 按 Commit changes。
6. 等待約 1–3 分鐘，GitHub Pages 會自動更新網站。

## 商品更新原則

未來新增商品時，請準備：
- 商品圖片
- 品牌
- 商品名稱
- 售價
- 尺寸
- 顏色
- 簡短介紹
