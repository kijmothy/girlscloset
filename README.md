# 女孩衣櫃間 Girls Closet 官網 V2 正式版

這是女孩衣櫃間 Girls Closet 的 GitHub Pages 靜態官網檔案。

## 檔案說明

- index.html：網站內容
- style.css：網站樣式
- logo.png：品牌 Logo
- README.md：說明文件

## 更新 GitHub Pages

1. 下載 ZIP 並解壓縮。
2. 到 GitHub repository：girlscloset。
3. 點 Add file → Upload files。
4. 上傳並覆蓋 index.html、style.css、logo.png、README.md。
5. 按 Commit changes。
6. 等待約 1–3 分鐘，GitHub Pages 會自動更新網站。
