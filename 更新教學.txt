Girls Closet v3.0 商品上架教學

一、上架一件新商品

1. 把商品圖片放進資料夾：
images/kuose/
images/mishow/
images/snidel/
images/fint/

2. 打開 products.js

3. 複製一整段商品資料，例如：
{
  id: "kuose-dress-001",
  brand: "KUOSE",
  name: "法式學院風洋裝",
  price: "NT$ 請洽客服",
  size: "S / M / L",
  color: "依品牌供應",
  material: "依品牌標示為準",
  description: "溫柔學院感設計，適合日常、約會與輕正式場合。",
  image: "",
  badge: "NEW",
  soldout: false
}

4. 修改內容。
如果要放圖片，image 請填：
image: "images/kuose/first-bloom.jpg"

二、標記售完

把：
soldout: false

改成：
soldout: true

三、改品牌篩選

目前首頁篩選品牌為：
KUOSE / MISHOW / SNIDEL / Fi.n.t

如果未來新增品牌，需要同時修改 index.html 的 filter-button。

四、LINE 連結

目前全站 LINE 連結：
https://line.me/ti/p/Fj1bSn35pG

五、GitHub 上傳

第一次升級 v3.0：整包上傳。
之後新增商品：通常只上傳 products.js 和新圖片即可。
