Girls Closet 官網更新教學 v1.1

一、替換商品圖片
把圖片放進 images/products/ 對應品牌資料夾。
例如：images/products/kuose/kuose01.jpg

二、修改商品卡
在 index.html 搜尋 product-card，找到對應商品後修改：
- 商品名稱
- 價格
- 尺寸
- 簡介

三、修正 LINE 連結
如果之後有 LINE 官方帳號連結，可以把 href="#contact" 改成正式 LINE 網址。

四、上傳 GitHub
改好後到 GitHub → Add file → Upload files → Commit changes。
