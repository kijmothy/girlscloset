# 女孩衣櫃間 Girls Closet 官網

這是一個可用於 GitHub Pages 發布的靜態網站。

## 檔案說明

- index.html：網站主要內容
- style.css：網站樣式
- README.md：說明文件

## 發布到 GitHub Pages

1. 到 GitHub 建立一個新的 repository
2. 將 index.html、style.css、README.md 上傳
3. 進入 Settings → Pages
4. Source 選擇 Deploy from a branch
5. Branch 選 main / root
6. 儲存後等待 GitHub 產生網站連結
